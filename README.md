# Panta Rhei Books

[![Deploy](https://github.com/Panta-Rhei-Books/website/actions/workflows/pages.yml/badge.svg)](https://github.com/Panta-Rhei-Books/website/actions/workflows/pages.yml)

Source code for [panta-rhei-books.org](https://panta-rhei-books.org) — the official website of the *Panta Rhei* seven-volume research book series.

## The Series

| Vol. | Title |
|------|-------|
| I | Categorical Foundations |
| II | Categorical Holomorphy |
| III | Categorical Spectrum |
| IV | Categorical Microcosm |
| V | Categorical Macrocosm |
| VI | Categorical Life |
| VII | Categorical Metaphysics |

A unified mathematical framework spanning category theory, complex analysis, spectral theory, quantum structure, cosmology, the mathematics of life, and metaphysics.

## Development

```bash
bundle install
bundle exec jekyll serve --livereload
```

Open [http://127.0.0.1:4000](http://127.0.0.1:4000).

## Architecture

- **Jekyll 4.4** static site generator
- **GitHub Pages** deployment via GitHub Actions
- Pure HTML / CSS / vanilla JS — no frameworks
